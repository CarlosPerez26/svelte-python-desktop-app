# Aplicación de Escritorio SvelteKit + Python

Este es el proyecto raíz para una aplicación de escritorio de alta performance, utilizando **FastAPI** como backend y **SvelteKit** como frontend.

## 🚀 Inicio Rápido

1.  **Backend:**
    ```bash
    cd backend
    ./venv/bin/python3 -m uvicorn main:app --reload
    ```
2.  **Frontend:**
    ```bash
    cd frontend
    bun run dev
    ```

## 📂 Estructura del Proyecto

-   `/frontend`: Interfaz SvelteKit (Svelte 5 + Tailwind).
-   `/backend`: API FastAPI y lógica de negocio en Python.
-   `/docs`: Documentación detallada del proyecto.
-   `/.agents/skills`: Estándares y habilidades del agente de desarrollo.
